MONITORING
==========

Check GitHub Actions:
1. Go to repo → Actions tab
2. Click "JJFO Daily News Scraper"
3. View recent runs
4. Check for failures
5. Read logs if failed

Check Google Sheets:
1. Go to Dispatch_Logs sheet
2. Verify daily entries
3. Check status is "Success"
4. Note yield (# articles)

Check Email:
1. Verify arrives at 8 AM IST
2. Check format is correct
3. Verify links work
4. Check for relevant articles

SCALING
=======

If you want to scale:
- Add more companies: Just add to Google Sheet
- Add more sources: Edit SOURCES in scraper_v2.py
- Change time: Edit cron in daily-scrape.yml
- Add more recipients: Add to Email_Recipients sheet
- Customize categories: Edit keywords in scraper_v2.py

All changes are free and automatic!
