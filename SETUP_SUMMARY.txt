JJFO NEWS BULLETIN - SETUP SUMMARY
===================================

PROJECT STRUCTURE
=================
jjfo-news-bulletin/
├── .github/workflows/daily-scrape.yml  (GitHub Actions - 8 AM IST daily)
├── scraper.py                          (Main Python scraper)
├── index.html                          (Admin dashboard)
├── companies_data.json                 (60+ companies with competitors)
├── requirements.txt                    (Python dependencies)
├── README.md                           (Documentation)
├── CREDENTIALS.md                      (Setup guide)
└── .gitignore                          (Git ignore)

WHAT'S BEEN BUILT
=================

1. Python Scraper (scraper.py)
   - Connects to Google Sheets
   - Queries NewsAPI for all companies
   - Categorizes news by priority
   - Deduplicates using headline similarity
   - Sends email via Resend.dev
   - Logs to Google Sheets

2. GitHub Actions Workflow (daily-scrape.yml)
   - Runs daily at 8:00 AM IST (2:30 AM UTC)
   - Installs dependencies
   - Executes scraper
   - Logs results

3. Admin Dashboard (index.html)
   - Manage portfolio companies
   - Manage email recipients
   - View dispatch logs
   - GitHub Pages deployment

4. Company Data (companies_data.json)
   - 60+ portfolio companies
   - Auto-detected competitors
   - Pre-defined sectors

5. Google Sheets Database
   - Companies tab
   - Email_Recipients tab
   - News_Items tab
   - Dispatch_Logs tab

YOUR ACTION ITEMS
=================

Step 1: Create Service Accounts (15 minutes total)
---
A. NewsAPI
   - Go to: https://newsapi.org/
   - Sign up → Get API Key
   - Copy key

B. Resend.dev
   - Go to: https://resend.com/
   - Sign up → Create API Key
   - Copy key

C. Google Cloud
   - Go to: https://console.cloud.google.com/
   - Create project: "JJFO News Bulletin"
   - Enable Google Sheets API
   - Create Service Account
   - Download JSON credentials
   - Share Google Sheet with service account email

Step 2: Create Google Sheet (5 minutes)
---
1. Go to: https://sheets.google.com/
2. Create new sheet: "JJFO News Bulletin"
3. Create tabs:
   - Companies (columns: Company Name, Sector, Competitors)
   - Email_Recipients (columns: Email)
   - News_Items (columns: Headline, URL, Source, Company, Category, Published_At)
   - Dispatch_Logs (columns: Timestamp, Status, Yield, Details)
4. Add initial data from companies_data.json

Step 3: Set Up GitHub Repository (5 minutes)
---
1. Go to: https://github.com/
2. Create new private repository: "jjfo-news-bulletin"
3. Clone and push this code
4. Go to Settings → Secrets and variables → Actions
5. Add these secrets:
   - NEWSAPI_KEY: [Your NewsAPI key]
   - RESEND_API_KEY: [Your Resend API key]
   - GOOGLE_SHEETS_ID: [Your Sheet ID from URL]
   - GOOGLE_CREDENTIALS: [Your Google Service Account JSON as string]

Step 4: Enable GitHub Pages (2 minutes)
---
1. Go to repository Settings
2. Scroll to "Pages"
3. Select "Deploy from a branch" → "main"
4. Dashboard will be at: https://[username].github.io/jjfo-news-bulletin/

Step 5: Configure Domain (Optional, 5 minutes)
---
1. Go to your domain registrar
2. Update DNS:
   - Add CNAME: app.jjfo.in → [username].github.io
   - Or update A records to GitHub Pages IPs
3. Wait 24-48 hours for propagation

TROUBLESHOOTING
===============
Email not sending?
- Verify Resend API key is correct
- Check investment@jjfo.com is in Email_Recipients sheet
- Check GitHub Actions logs for errors

No news scraped?
- Verify NewsAPI key is correct
- Check company names match exactly in Google Sheets
- Verify NewsAPI free tier not exceeded

GitHub Actions not running?
- Check cron schedule in .github/workflows/daily-scrape.yml
- Verify all secrets are set correctly
- Check GitHub Actions is enabled
