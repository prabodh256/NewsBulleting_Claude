╔══════════════════════════════════════════════════════════════════╗
║     JJFO COMPETITOR TRACKER — GO-LIVE GUIDE (First-time user)   ║
╚══════════════════════════════════════════════════════════════════╝

Total time: ~25 minutes. You need:
  • A browser (Chrome / Edge)
  • Your Resend account (or create one for free)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — Get your Resend API key   (5 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open https://resend.com in your browser
2. Click "Sign up" → enter your email and create a password
3. Confirm your email (check your inbox)
4. Once logged in, click "API Keys" in the left sidebar
5. Click "Create API Key"
6. Give it a name (e.g. "jjfo tracker") → click Create
7. COPY the key that appears (it starts with "re_")
   ⚠️  You can only see this key ONCE — copy it now!
   Keep it in a notepad for the next step.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — Create your .env file   (2 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
In the project folder you'll find a file called: .env.example

1. Open it with Notepad (right-click → Open with → Notepad)
2. Fill in your values:

   RESEND_API_KEY=re_paste_your_key_here
   EMAIL_FROM=onboarding@resend.dev
   FLASK_SECRET_KEY=jjfotracker2024xyzABC

   Note: EMAIL_FROM = "onboarding@resend.dev" works for testing.
   Once you add your own domain in Resend, change it to: noreply@jjfo.in

3. Save As → change filename to: .env  (remove ".example")
   ⚠️  Make sure it is saved as ".env" not ".env.txt"

   How to check: In File Explorer, enable "File name extensions"
   (View → Show → File name extensions). The file must show as ".env".


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — Create a free PythonAnywhere account   (3 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open https://www.pythonanywhere.com in your browser
2. Click "Pricing & Signup" → choose "Create a Beginner account" (FREE)
3. Choose a username — this becomes your URL, e.g.:  jjfonews
   Your dashboard will be at: https://jjfonews.pythonanywhere.com
4. Enter email + password → Create account
5. Log in


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4 — Upload your files   (5 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
You are now on PythonAnywhere. Click "Files" in the top menu.

You'll see you are in folder: /home/jjfonews/

Upload these files ONE BY ONE using the "Upload a file" button:
  ✓ agent.py
  ✓ app.py
  ✓ scraper.py
  ✓ wsgi.py
  ✓ requirements.txt
  ✓ .env                 ← the one you created in Step 2
  ✓ companies.csv
  ✓ recipients.csv
  ✓ dispatch_log.csv

Now upload the templates folder:
  1. Click "New directory" → type: templates → Create
  2. Click on the "templates" folder to open it
  3. Upload these 4 files from your "templates" folder:
       ✓ base.html
       ✓ index.html
       ✓ add.html
       ✓ recipients.html
  4. Click the folder icon "← Up" to go back to /home/jjfonews/


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5 — Install Python packages   (3 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Click "Consoles" in the top menu
2. Click "Bash"  (opens a black terminal window)
3. Type this command exactly and press Enter:

   pip3.10 install flask requests beautifulsoup4 feedparser resend python-dotenv --user

4. Wait ~1 minute. You'll see packages installing.
5. When you see the "$" prompt again, it's done.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 6 — Set up the web app   (5 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Click "Web" in the top menu
2. Click "Add a new web app" → Next
3. Select "Manual configuration" (NOT Flask) → Next
4. Select "Python 3.10" → Next → Next

5. You'll see a page with your web app settings.
   Find the section "Code" → click the link next to "WSGI configuration file"
   It opens a text editor.

6. SELECT ALL the existing text in the editor (Ctrl+A) → DELETE it

7. TYPE (or copy-paste) these 9 lines:

   import sys
   import os
   project_home = '/home/YOUR_USERNAME'
   if project_home not in sys.path:
       sys.path.insert(0, project_home)
   os.chdir(project_home)
   from app import app as application

   ⚠️  IMPORTANT: Replace YOUR_USERNAME with your actual username
       e.g.:  project_home = '/home/jjfonews'

8. Click "Save" (top right of the editor)
9. Click the back arrow "←" to go back to the Web app page
10. Click the green "Reload" button (big button at the top)

11. Click the link at the top of the page (your URL):
    https://jjfonews.pythonanywhere.com
    You should see the dashboard! 🎉


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 7 — Add email recipients   (1 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open your dashboard: https://jjfonews.pythonanywhere.com
2. Click "✉️ Recipients" in the top right
3. Type each email address → click "+ Add"
4. Repeat for all email addresses that should receive the digest


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 8 — Set up the daily 2:30 AM scraper   (2 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. On PythonAnywhere, click "Tasks" in the top menu
2. You'll see "Scheduled tasks"
3. In the "Daily" section, set the time to:  02:30  UTC
4. In the "Command" box, type:
   python3.10 /home/YOUR_USERNAME/scraper.py
   (replace YOUR_USERNAME with your username, e.g. jjfonews)
5. Click "Create"

✓ Done! The scraper will now run every day at 2:30 AM GMT (= 8:00 AM IST).


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 9 — Test everything right now   (2 min)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open your dashboard: https://jjfonews.pythonanywhere.com
2. Click "▶ Run Scraper Now"
3. Wait ~1 minute (it scrapes 15+ sources)
4. You'll see a message: "Scraper ran: ..."
5. Check the inbox of your recipient emails — the digest should arrive!

If the email didn't arrive, check:
  • Spam folder
  • That your .env has the correct RESEND_API_KEY
  • That you added at least one recipient in Step 7

HOW TO ADD A NEW COMPANY (any time, from any device)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open https://jjfonews.pythonanywhere.com
2. Click "+ Add Company"
3. Type the company name → click 🔍 Search
4. The agent shows 3-5 options with descriptions
   → Pick the correct one (e.g. "Paytm — Indian fintech company")
5. Click "✓ Confirm & Find Competitors"
6. The agent auto-fills the sector and competitors — done!
   The company will be tracked in tomorrow's 2:30 AM email.

TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Problem: Website shows "502 Bad Gateway" or "Something went wrong"
Fix:     PythonAnywhere → Web tab → click Error log
         Look for the red error line and fix the issue.
         Most common cause: wsgi.py has wrong username.

Problem: "No module named flask" error
Fix:     Re-run Step 5 (pip install command)

Problem: Email not received
Fix:     • Check spam folder
         • PythonAnywhere → Files → open dispatch_log.csv
           The last line shows: timestamp, status, article_count, recipient_count
           If status = "email-failed", your RESEND_API_KEY is wrong.
         • Make sure recipients.csv has at least one email address

Problem: Website works but "Run Scraper Now" does nothing
Fix:     PythonAnywhere → Consoles → Bash → type:
         cd /home/YOUR_USERNAME && python3.10 scraper.py
         This shows the exact error.
