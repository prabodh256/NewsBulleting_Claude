STEP 2: Create GitHub Repository (3 minutes)
=============================================
1. Go to: https://github.com/
2. Click "+" → "New repository"
3. Name: "jjfo-news-bulletin"
4. Select "Private"
5. Click "Create repository"
6. You'll see clone instructions

STEP 3: Clone & Add Files (5 minutes)
====================================
1. Clone the repo to your computer:
   git clone https://github.com/[YOUR_USERNAME]/jjfo-news-bulletin.git
   cd jjfo-news-bulletin

2. Copy these files from /home/ubuntu/jjfo-news-bulletin/:
   - scraper.py (UPDATED with 16 sources)
   - requirements.txt
   - companies.csv
   - recipients.csv

   - .github/workflows/daily-scrape.yml
   - .gitignore

3. Push to GitHub:
   git add .
   git commit -m "Add JJFO News Bulletin with 16 sources"
   git push

STEP 4: Add GitHub Secret (2 minutes)
====================================
1. Go to your GitHub repository
2. Click "Settings" (top right)
3. Click "Secrets and variables" → "Actions"
4. Click "New repository secret"
5. Add this secret:
   - Name: RESEND_API_KEY
   - Value: [Your Resend API key from Step 1]
   - Click "Add secret"

TROUBLESHOOTING
===============

Email not received?
- Check GitHub Actions logs (Actions tab → latest run)
- Verify Resend API key is correct
- Check recipients.csv has correct emails
- Check spam folder
- Note: Scraping 16 sources takes 2-3 minutes, so email may be delayed

No articles found?
- Check company names match exactly in companies.csv
- Check RSS feeds are working (visit URLs manually)
- Check GitHub Actions logs for errors
- First run may take longer as it scrapes all sources

GitHub Actions not running?
- Check cron schedule in .github/workflows/daily-scrape.yml
- Should be: "30 2 * * *" (8 AM IST)
- Verify all secrets are set correctly
- Check GitHub Actions is enabled

Scraper timing out?
- Scraping 16 sources takes 2-3 minutes
- GitHub Actions has 35-minute timeout, so this is fine
- If timeout occurs, check internet connection or source availability

CUSTOMIZATION
=============

Change time?
- Edit .github/workflows/daily-scrape.yml
- Change cron: "30 2 * * *" to your desired time
- Format: "MM HH * * *" (minutes hours * * *)
- Example: "0 9 * * *" = 9:00 AM UTC

Add more companies?
- Edit companies.csv
- Add rows: Company Name, Sector, Competitors
- Commit and push

Add more recipients?
- Edit recipients.csv
- Add emails (one per line)
- Commit and push

Change email format?
- Edit scraper.py function format_email()
- Modify the body variable
- Commit and push

Add more news sources?
- Edit scraper.py
- Add to RSS_FEEDS or SOURCES dictionaries
- Commit and push
